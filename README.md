#API Execute Code

API to execute code and return whether code result in error ou success.

**request**

- Method: POST.
- Params: code, lang.


**return:**

```json

	{"status": "error"};

```

```json

	{"status": "noerror"}

```


**Language supported**

- C

- Python

**Requirements**

Interpreter or Compiler of languages.
