# API Execute Code



> API to execute code and return whether code result in error ou success.

**Request**

- Method: POST.
- Params: code, lang.


**Return:**

```json

{"status": "error"};

```

```json

{"status": "noerror"}

```


**Language supported**

- C

- Python

**Requirements**

Interpreter or Compiler of languages.
